# Artifact Types — atom-activity-generate

## single classroom activity

The single artifact produced by this atom is **one classroom activity** — a structured JSON object
with procedure, materials, timing, DOK level, and embedded differentiation supports.

**Required fields:**
- `activity`: `{title, type, duration_minutes, dok_level, procedure, materials, ell_support, iep_support}`
- `human_review_required: true`

**Not produced by this atom:** full lesson plans, assessment items, or differentiated rewrites.
