# Artifact Types — atom-misconception

## misconception list

The single artifact produced by this atom is a **list of common misconceptions** for a topic —
a structured JSON array, each entry with the misconception, why it occurs, and an instructional remedy.

**Required fields:**
- `misconceptions`: list of `{misconception, why_it_occurs, instructional_remedy}`
- `human_review_required: true`

**Not produced by this atom:** activities, assessments, or standards lookups.
