- **Linked standard:** `CCSS.MATH.CONTENT.3.OA.C.7` — fluently multiply within 100 (CCSS-Math, 2010)

## Dosage
- Frequency: 4×/week · Duration: 15 min · Group size: 3

## Intervention routine
1. 1-minute warm-up of mastered facts (build confidence).
2. Explicit strategy for a target set (e.g., ×3 via "double + one group").
3. Paired practice with flashcards + immediate corrective feedback.
4. 1-minute timed check; chart the score with the student.

## Progress monitoring
- **Measure:** 1-minute multiplication-fact CBM (digits correct) · **Cadence:** weekly
- **Aim:** from baseline ~18 to 40 digits correct in 8 weeks.
