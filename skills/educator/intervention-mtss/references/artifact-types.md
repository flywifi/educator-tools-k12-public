# artifact-types.md
## Artifacts produced by intervention-mtss

All outputs are team decision support with placeholder data and the metadata block.

### Tiered intervention plan (Tier 2 / Tier 3)
- **Purpose:** a targeted (T2) or intensive (T3) plan for a precise skill.
- **Required elements:** target skill; tier; **dosage** (frequency × minutes × group size); the
  intervention routine/steps; materials; progress-monitoring measure + cadence; decision rules.
- **Template:** `assets/templates/intervention-plan-template.md`.

### MTSS documentation
- **Purpose:** record for a problem-solving / data team meeting.
- **Required elements:** concern + baseline data (placeholder); tier history; intervention + fidelity;
  progress data summary; team decision + next review date.

### Progress-monitoring schedule
- **Purpose:** plan and record measurement over time.
- **Required elements:** the measure (e.g., CBM), cadence (e.g., weekly), goal line/aim, a recording
  table (placeholder), and the decision rule (e.g., 4 points below the aim line → intensify).

### Data-based decision note
- **Purpose:** summarize what the data says and recommend continue / intensify / fade / refer.
- **Required elements:** the data trend, the rule applied, and the recommendation — framed for the
  team, never an eligibility determination.

## Validation (extends `shared/quality/verification-checklists.md`)
- Target skill is specific and measurable; tier matches intensity/dosage.
- A progress-monitoring measure, cadence, and decision rule are present.
- Tied to a real standard; placeholders only; MTSS not conflated with special-ed eligibility.
