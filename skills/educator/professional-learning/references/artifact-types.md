# artifact-types.md
## Artifacts produced by professional-learning

Non-evaluative, growth-oriented, grounded in adult learning; placeholders + metadata block.

### Observation / look-for tool
- **Purpose:** focus a coaching observation on specific, observable practice.
- **Required elements:** the instructional focus; 3-6 **observable** look-fors (not judgments);
  evidence space; reflection/coaching prompts; a "glow + grow + next step" structure.
- **Template:** `assets/templates/observation-tool-template.md`.

### Coaching conversation guide
- **Purpose:** structure a non-evaluative coaching conversation.
- **Required elements:** opening (asset-based); evidence to discuss; reflective questions; a
  teacher-owned goal + next step; follow-up plan.

### PD session plan
- **Purpose:** plan a professional development session.
- **Required elements:** objective; agenda with timing; active-learning structures (not lecture-only);
  modeling + practice + reflection; materials; a follow-up/transfer plan.

## Validation (extends `shared/quality/verification-checklists.md`)
- Look-fors are **observable**, not evaluative judgments.
- Tone is asset-based and growth-oriented (coaching, not gotcha).
- PD uses active learning (modeling, practice, reflection), not lecture-only.
- Placeholders only; no real teacher/student data.
