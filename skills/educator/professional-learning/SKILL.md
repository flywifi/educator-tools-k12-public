# professional-learning

Creates coaching and PD resources grounded in adult-learning principles. Coaching here is
**non-evaluative and growth-oriented** (distinct from administrator evaluation).

## 1. Follow the pipeline (`references/method.md`)
Within Generation: `Analysis → (alignment to a practice/standard) → Differentiation → Generation`.

1. **Analysis** — identify the artifact (`references/artifact-types.md`), the **instructional focus**
   (e.g., "checks for understanding", "discussion techniques"), and the audience's experience level.
   Log assumptions.
2. **Alignment** — anchor look-fors to concrete, observable teacher practice (and, where relevant, a
   teaching framework or the student standards the practice serves).
3. **Adult-learning design** — apply `references/adult-learning.md`: relevance, choice, active
   practice, reflection; non-evaluative, asset-based language.
4. **Generation** — draft from the template: clear focus, observable indicators, reflection/coaching
   prompts, and next steps.

## 2. Validate, then gate
Run the universal + PD checks (`shared/quality/verification-checklists.md`); self-score against
`references/quality-gates.md`, then hand to **quality-review**.

## 3. Always emit the metadata block
Per `protocol-layer/metadata-schema.md`, with `human_review_required: true`. Placeholders only (no real
teacher or student data).

Artifact types: `references/artifact-types.md`. Template:
`assets/templates/observation-tool-template.md`.
