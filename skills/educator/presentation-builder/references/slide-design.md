# slide-design.md
## Instructional slide design principles
What makes a teaching deck effective and accessible (feeds Educational Quality + Accessibility gates).

## 1. One idea per slide
Each slide carries a single point. Split dense content; never paste a worksheet onto a slide. Aim for
few words + a supporting visual; the teacher narrates the rest.

## 2. Mirror gradual release
Sequence the deck like the lesson: hook → objective → model (I do) → guided (we do) → independent
(you do) → closing/recap. Make transitions between phases explicit.

## 3. Show, don't just tell
Use models, diagrams, and worked examples (concrete → representational → abstract for math/science).
Visuals must carry meaning and include **alt text** for accessibility.

## 4. Readable and accessible
Large text; high contrast; limited text per slide; grade-appropriate vocabulary
(`shared/quality/readability-age.md`). Support UDL: pair visuals with words; cue what to attend to.

## 5. Build in interaction
Add check-for-understanding slides (turn-and-talk prompt, quick question, poll). A deck that's all
delivery and no checks scores low on Educational Quality.

## 6. Keep it honest and safe
No fabricated facts or images presented as real; no real student data; inclusive, bias-aware
examples. Render with the `pptx` skill — don't hand-build the file format.
