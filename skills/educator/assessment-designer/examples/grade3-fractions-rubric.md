# Rubric: Explaining Unit Fractions (1/b)

> Worked reference output, aligned to the gold lesson
> `skills/lesson-planner/examples/grade3-fractions-lesson.md` (same standard) so a lesson and its
> scoring tool fit together. Evaluated by quality-review → Approved.

- **Grade band / Subject:** 3-5 (Grade 3) / Mathematics
- **Standard:** `CCSS.MATH.CONTENT.3.NF.A.1` — unit fractions 1/b from equal partitioning
  (CCSS-Math, 2010).
- **What it scores:** a student's explanation + model of a unit fraction.

| Criterion | 4 — Exemplary | 3 — Proficient | 2 — Developing | 1 — Beginning |
|---|---|---|---|---|
| Equal partitioning | Partitions into b **equal** parts and explains why equality matters | Partitions into b equal parts | Partitions but parts are unequal | Does not partition meaningfully |
| Naming the unit fraction | Correctly names 1/b and connects denominator to # of equal parts | Correctly names 1/b | Names a fraction but mislabels parts | Cannot name the fraction |
| Representation | Uses a clear model (area/number line) accurately | Uses a model with minor errors | Model present but unclear | No usable model |
| Math language (MP6) | Precise: *equal parts, whole, denominator, unit fraction* | Mostly precise | Some vocabulary, imprecise | Little/no math vocabulary |

- **Student-friendly version:** "4 = I split it into equal parts, named 1/b, drew it right, and used
  fraction words. 1 = I'm just starting."
- **Scoring note:** formative use — totals guide reteach grouping, not a letter grade.

## Using the data
14-16 → ready for a/b (multiple parts). 9-13 → targeted practice on equal partitioning. ≤8 → Tier 2
with concrete tiles.
