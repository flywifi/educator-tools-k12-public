# Example — Accommodation plan (DRAFT) for a Grade-3 reading need

> Worked reference output. Decision support for the IEP/504 team; placeholder student; validate
> against the actual plan and policy. Evaluated by quality-review → Approved.

- **Student:** Student A (placeholder)
- **Grade band / Subject:** 3-5 (Grade 3) / Reading & across content areas
- **Area of need:** decoding/reading fluency affecting access to grade-level text

## Accommodations (expectation unchanged)
| Category | Accommodation | Where / when |
|---|---|---|
| Presentation | text-to-speech for grade-level passages; read-aloud of directions | reading, science, social studies |
| Response | allow oral responses / speech-to-text for written items | all content areas |
| Setting | reduced-distraction space for assessments | testing |
| Timing | extended time (1.5x) on reading-heavy tasks | reading, assessments |
| Organization | graphic organizers; chunked directions | all content areas |

## Notes
- The student is still held to **grade-level comprehension expectations** — these supports change
  *access*, not the standard. A change to the comprehension expectation itself would be a
  *modification* and a separate team decision.
- Pairs with EL/UDL supports if applicable (`shared/differentiation/`).
