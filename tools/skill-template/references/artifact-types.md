# artifact-types.md
## Artifacts produced by this skill

Replace this file with the specific artifact types this skill generates. For each artifact, define:

- **Purpose** — what it is and when it's used.
- **Required elements** — the parts every instance must contain.
- **Standards** — which framework(s) typically apply and how they're cited.
- **Differentiation** — the UDL / tiering / EL / IEP supports expected by default.
- **Template** — pointer to the matching file under `assets/templates/`.
- **Validation** — the artifact-specific checks (extends `shared/quality/verification-checklists.md`).
