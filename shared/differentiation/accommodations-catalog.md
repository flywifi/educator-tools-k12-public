# accommodations-catalog.md
## Differentiation Engine — Accommodations & Modifications Catalog
Canonical source. A reference catalog for IEP/504 supports. **Critical distinction:**

- **Accommodation** — changes *how* a student accesses content or shows learning; the **learning
  expectation is unchanged** (e.g., extended time, text-to-speech).
- **Modification** — changes *what* a student is expected to learn; the **expectation is changed**
  (e.g., fewer/alternate objectives). Modifications are significant decisions — flag for the team.

> **Legal boundary.** This catalog is a menu of *common, generic* supports. It is **not** a
> substitute for a student's actual IEP/504 plan or for team/legal judgment. Outputs that reference
> accommodations must set `human_review_required: true` and state that supports must match the
> student's real plan and local/state policy (SECURITY_AND_SAFETY.md). **Never use real student
> data** — placeholders only.

---

## 1. Accommodation categories (examples)

- **Presentation:** read-aloud / text-to-speech, large print, audio, visual supports, simplified
  directions, fewer items per page.
- **Response:** speech-to-text, scribe, oral response, typed instead of handwritten, use of a
  calculator/manipulatives where not the skill assessed.
- **Setting:** small group, reduced-distraction space, preferential seating.
- **Timing/Scheduling:** extended time, frequent breaks, tasks split across sessions.
- **Organization/Support:** checklists, graphic organizers, exemplars, cueing, chunking.

## 2. Modification categories (examples)
- Reduced number/complexity of objectives; alternate (linked) standards; adjusted grading against
  modified expectations; alternate assessments. Always team-decided and documented.

## 3. How skills use this catalog
During pipeline step 3 (Differentiation), when a request involves IEP/504 supports, select
appropriate **accommodations by default** (expectation-preserving) and treat **modifications** as
team decisions to be surfaced, not assumed (assumptions-protocol.md → escalate). Record what was
applied in metadata (`differentiation`). The `special-education-support` skill (Phase A+) builds on
this catalog.

## 4. Florida
